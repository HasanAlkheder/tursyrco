TurSyrCo: A Large-Scale Turkish–Syrian Arabic Parallel Corpus
This repository contains the official code, lexicon, and sample data for the paper: "Machine Translation between Turkish and Syrian Arabic: Introducing TurSyrCo, a Large-Scale Professionally Curated Parallel Corpus."

📌 Overview
TurSyrCo is a professionally curated parallel corpus comprising 50,000 sentence pairs derived from dubbed television series. It is designed to address the severe resource scarcity for Turkish–Syrian Arabic machine translation, capturing authentic conversational Syrian Arabic across diverse genres.

Key Features:
Size: 50,000 high-quality, human-translated sentence pairs.

Source: Professional dubbing scripts (NiS Studios) ensuring dialectal authenticity.

Domain: Conversational, emotional, and everyday discourse (TV Series).

Included Resources: Syrian dialect lexicon and NMT fine-tuning configurations.
